## üé® Design & Styling

### Color System
- **Primary**: Blue (#2563eb) - CTAs, highlights
- **Base**: Zinc (neutral grays) - Text, backgrounds
- **Accent**: Red (#dc2626) - Available for logo/branding
- **Overlays**: Black with 40% opacity

### Typography
- **Font**: Inter (configured in layout.tsx)
- **Responsive sizes**: text-sm to text-7xl
- **Weights**: Regular (400), Semibold (600), Bold (700)

### Responsive Breakpoints
- **sm**: 640px - Mobile/tablets
- **md**: 768px - Tablets
- **lg**: 1024px - Desktop
- **xl**: 1280px - Large desktop

### Spacing System
- Tailwind default 4px scale
- Padding: px-4 to px-12
- Gaps: gap-4 to gap-8
- Margins: my-4 to my-8
